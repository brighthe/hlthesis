# 摘要

&emsp;&emsp;近年来，机器学习增强的快速结构分析与结构设计方法受到了广泛关注。然而，在大多数相关研究中，机器学习模型的泛化能力以及训练数据集生成的巨大成本，是最常受到批评的两个方面。本文结合了子结构方法的普适性优势与算子学习架构的优越预测能力。具体而言，本文利用一种新的基于力学机理的损失函数，在无需预先制备数据集的情况下，成功训练了一个轻量级神经网络映射，用于建立子结构内部材料分布与相应连续多尺度形函数之间的关系。由此，本文提出了一种问题无关机器学习模型，即 PIML，该模型能够普遍适用于任意尺寸、不同边界条件的大规模结构的高效线弹性分析与设计优化。多个算例验证了本文方法在提升计算效率以及处理不同类型优化问题方面的有效性。该基于 PIML 模型的设计优化框架还可进一步扩展到大规模多物理场问题中。

# 1 引言

&emsp;&emsp;结构分析与拓扑优化已经广泛应用于工程领域（Rozvany，2009；Deaton 和 Grandhi，2014；Guo 和 Cheng，2010；Sigmund 和 Maute，2013）。然而，尽管拓扑优化为设计具有最优性能的结构提供了一种合理方法，其求解过程通常需要对一系列中间结构进行数十次到数百次迭代数值分析；对于大规模优化问题而言，这部分计算可能占据超过 $95\%$ 的总计算成本。因此，提高大规模结构数值分析的效率，对于推动拓扑优化的研究前沿与工程应用具有关键意义。已有文献中，并行算法（Borrvall 和 Petersson，2001；Aage 和 Lazarov，2013；Aage 等，2017；Liu 等，2018a）、多重网格方法（Amir 等，2014；Hackbusch，2013；Le Maitre 等，2003）、冗余自由度移除技术（Zhang 等，2017；Du 等，2022）以及降阶建模方法（Amsallem 等，2015；Hoang 等，2016；Dutta 等，2018；Xiao 等，2020）已经被用于提升拓扑优化的求解效率。

&emsp;&emsp;随着人工智能的快速发展，机器学习增强的拓扑优化方法近年来受到了广泛关注。Sosnovik 和 Oseledets 在 SIMP 框架中使用卷积神经网络，仅需少量迭代步即可获得最终优化设计（Sosnovik 和 Oseledets，2019）。为了进一步实现实时拓扑优化，Yu 等直接构造了一个卷积神经网络模型，用于建立设计信息到最终优化结构之间的映射关系（Yu 等，2019）。随后，Abueidda 等利用基于残差学习的 Res-U-Net 网络对这类工作进行了改进，显著提升了预测模型在超弹性结构优化中的性能（Abueidda 等，2020）。Lei 等则在移动可变形组件 MMC 框架中使用支持向量机，将外载荷的位置与方向映射到最终优化构型中组件的几何参数（Lei 等，2019）；该方法后来又分别采用 Res-U-Net 网络（Zheng 等，2021）和卷积神经网络（Geng 等，2023）进行了改进。更多以最优设计为输出的端到端机器学习模型，可参见 Woldseth 等（2022）的综述或相关讨论。

&emsp;&emsp;与图像识别（Shafiq 和 Gu，2022）和智能翻译（Cho 等，2014）等领域不同，结构分析与优化过程包含大量专业知识。已有研究指出，简单地将机器学习技术用于构造结构响应或结构性能的全局代理模型存在显著局限性（Woldseth 等，2022）。其中最突出的挑战在于，现有代理模型通常具有问题相关性；对于训练样本中未包含的模型，其预测精度会显著下降。因此，机器学习技术也被用于加速有限元分析，以增强神经网络的适用性并保证训练模型的一般性，而不是一味追求端到端映射。例如，Keshavarzzadeh 等基于神经网络与低秩近似理论，提出了一种计算细网格响应的新方法（Keshavarzzadeh 等，2021）。Zhang 等将卷积神经网络与张量分解相结合，相较于传统张量分解和适当广义分解方法取得了更高精度（Zhang 等，2022）。Yue 等提出了一种有限元卷积神经网络，用于建立高分辨率有限元解到低分辨率有限元解之间的映射（Yue 等，2021）。Chi 等利用优化迭代过程中收集的训练数据，在多分辨率框架下提出了一种高效的拓扑优化在线机器学习方法（Chi 等，2021）；该方法随后进一步发展为离线训练、在线预测的形式。数值算例表明，这类方法能够处理分辨率达到数千万量级的大规模设计问题，并将结构分析效率提高约 $30$ 倍（Senhora 等，2022）。
